# 更新日志 v1.8.5

## Bug 修复

- **修复翻译完成提示框**：批量翻译和单文件翻译完成后正确弹出提示框，询问是否在编辑器中打开结果
- **修复编辑器文件删除按钮**：解决路径格式不匹配导致删除按钮失效的问题
- **修复上色分辨率问题**：上色后图片保持原始尺寸（需配合超分使用以获得更好效果）

## UI 改进

- **文件夹分组显示**：文件列表支持按文件夹分组，文件夹可折叠，显示文件数量
- **批量选择文件夹**：主页面和编辑器支持 Ctrl/Shift 多选文件夹
- **编辑器保留分组**：从主页面进入编辑器时保留文件夹分组信息

## 代码清理

- 删除测试脚本和临时文件

## 升级方法

已安装用户：双击 `步骤4-更新维护.bat` → 选择"完整更新"
